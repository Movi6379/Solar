# 2MW Solar Plant Performance Dashboard

A real-time data visualization tool for monitoring a 2MW solar photovoltaic plant using a 7-day operational dataset. 

##  Key Features
- **Performance Ratio (PR) Tracking**: Real-time monitoring of system efficiency.
- **Inverter Benchmarking**: Comparison of power output across multiple inverters.
- **Data Quality Assessment**: Checks for sensor gaps and communication outages.
- **Peak Power Analysis**: Identifies maximum generation peaks during solar noon.

##  Technology Stack
- **Language**: Python 3.9+
- **Framework**: Streamlit (Dashboarding)
- **Data Handling**: Pandas & NumPy
- **Visuals**: Plotly Express (Interactive Charts)
- **Deployment**: GitHub & Streamlit Community Cloud
